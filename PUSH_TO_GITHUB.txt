# After creating the repository on GitHub, run these commands:

# 1. Add the remote repository
git remote add origin https://github.com/Tahas13/minilang-compiler.git

# 2. Rename branch to main (GitHub default)
git branch -M main

# 3. Push to GitHub
git push -u origin main

# You'll be prompted to login to GitHub
# Use your GitHub username and Personal Access Token (not password)

# To create a Personal Access Token:
# 1. Go to: https://github.com/settings/tokens
# 2. Click "Generate new token (classic)"
# 3. Select scopes: repo
# 4. Copy the token and use it as password when prompted
